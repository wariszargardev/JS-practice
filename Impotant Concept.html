Hoisting says declare all variable at the top of your file/block

Constant
  • Whose value will never change
  • You can create a constant array: const cars = ["Saab", "Volvo", "BMW"];
  • You can change an element: cars[0] = "Toyota";
  • You can add an element: cars.push("Audi");
  • But you can NOT reassign the array: const cars = ["Saab", "Volvo", "BMW"];
    cars = ["Toyota", "Volvo", "Audi"]; // ERROR
